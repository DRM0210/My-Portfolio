## Image Requirements

- **Format**: PNG, JPG, JPEG, or WebP
- **Recommended size**: 800x600px or similar aspect ratio
- **File size**: Keep under 500KB for optimal performance
- **Content**: Homepage screenshot of each live project

## How to Add Screenshots

1. Take a screenshot of each project's homepage
2. Save it with the exact filename listed above
3. Place it in this folder (`src/assets/projects/`)
4. The images will automatically appear in the portfolio

If a screenshot is missing, a placeholder will be shown instead.
